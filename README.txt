# week2_Assignment-html-Css

FUTURETECH LEARNING HUB
WEB DEVELOPMENT PROJECT

PROJECT TITLE:
Build a Multi-Page Educational Website

DESCRIPTION:
This project is a responsive educational website created using HTML and CSS. It demonstrates the use of semantic HTML structure, hyperlinks, images, forms, and modern CSS styling techniques.

WEBSITE PAGES:
1. Home Page (index.html)
2. About Page (about.html)
3. Courses Page (courses.html)
4. Contact Page (contact.html)

FEATURES USED:
- HTML semantic tags (header, nav, main, section, article, footer)
- Internal navigation links between pages
- External links to learning resources
- Images with proper alt attributes
- Contact form
- CSS Flexbox and Grid layout
- Responsive design for mobile, tablet, and desktop
- Custom colors, borders, shadows, and rounded corners

EXTERNAL LINKS USED:
- https://www.w3schools.com
- https://developer.mozilla.org
- https://www.freecodecamp.org
